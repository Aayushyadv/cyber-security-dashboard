
# 🔐 Cyber Attack Insights Dashboard

## 📌 Project Overview
The **Cyber Attack Insights Dashboard** is an interactive Power BI project designed to analyze and visualize cybersecurity attack data.  

This dashboard helps monitor:
- Attack patterns
- Threat severity levels
- Protocol-based attacks
- Monthly packet activity
- Geo-location attack distribution
- Anomaly detection metrics

The project demonstrates practical skills in:
- Power BI Dashboard Development
- Data Visualization
- KPI Reporting
- Cybersecurity Analytics
- Business Intelligence Reporting

# 📊 Dashboard Features

## KPI Cards
The dashboard includes:
- Total Attack Types
- Average Anomaly Score
- Geo-location Count
- Total Anomaly Score

---

## 📈 Interactive Visualizations

### 🌍 Geo-location Attack Map
- Displays attack distribution by location
- Identifies high-risk regions

### 📋 Protocol Analysis Matrix
- Compares attack types across:
  - TCP
  - ICMP
  - UDP

### 📅 Monthly Packet Trend Analysis
- Tracks packet length variations over time
- Helps identify traffic spikes

### ⚠ Attack Type Analysis
- Visual comparison of:
  - DDoS
  - Malware
  - Intrusion attacks

### 📌 Scatter Plot Analysis
- Displays relationship between:
  - Packet Length
  - Anomaly Scores

### 📦 Treemap Visualization
- Shows protocol distribution by attack type

---

# 🎛 Filters & Slicers
Interactive filters include:
- Severity Level
- Attack Type
- Protocol
- Threat Category

---

# 🔍 Key Insights
- DDoS attacks showed the highest frequency
- TCP protocol had significant attack activity
- Certain months displayed increased packet traffic
- Geo-location analysis highlighted concentrated attack regions
- Anomaly scores helped detect suspicious activities
